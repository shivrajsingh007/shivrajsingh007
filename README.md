<details>
<summary>⚙️ Click to Setup the Snake Animation (2 minutes)</summary>

1. In your profile repo (`shivrajsingh007/shivrajsingh007`), create: `.github/workflows/snake.yml`
2. Paste this:

```yaml
name: Generate Snake Animation
on:
  schedule:
    - cron: "0 */12 * * *"
  workflow_dispatch:
jobs:
  generate:
    runs-on: ubuntu-latest
    steps:
      - uses: Platane/snk/svg-only@v3
        with:
          github_user_name: shivrajsingh007
          outputs: |
            dist/github-snake.svg
            dist/github-snake-dark.svg?palette=github-dark
      - uses: crazy-max/ghaction-github-pages@v3.1.0
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

3. Go to **Actions** tab → run the workflow manually once → done! 🐍
